# DataDog Client

Very simple [DataDog](http://www.datadoghq.com/) client.

## Installation via Composer

```json
{
    "require": {
        "rzajac/datadogclient": "0.5.*"
    }
}
```

## Documentation

[Documentation](docs/index.md)
